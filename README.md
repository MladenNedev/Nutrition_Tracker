#  Nutrition Tracker

A full-stack web application that allows users to log meals, view daily nutrient breakdowns, and visualize macronutrient ratios.

##  Tech Stack
- PHP 8.3 / MySQL 8 / Docker
- Twig templating engine
- Vite + SCSS + JavaScript (Chart.js)
- REST API architecture

##  Features
- Register and Login system (secure password hashing)
- Add meals and foods dynamically
- Daily dashboard with macronutrient charts
- SCSS modular styling compiled via Vite

##  Setup
```bash
docker compose up -d
npm install
npm run dev   # for frontend compilation
